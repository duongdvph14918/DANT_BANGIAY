Hướng dẫn chọn size giày đá banh chính hãng

Làm theo bảng hướng dẫn sau bạn sẽ có những số đo chân cụ thể, dựa vào đó canh đo theo bảng size chuẩn Giày Chính hãng NIKE, ADIDAS, PUMA, MIZUNO, KAMITO, ASICS..

Công thức đo Size Giày = Chiều dài chân + 1cm ( Trừ Hao bề ngang + vớ )
